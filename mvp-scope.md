# MVP Scope — IronLog React Native

## In Scope
- local-only app tanpa backend
- routine CRUD dasar
- exercise library lokal dengan search + filter muscle group
- start workout dari routine
- active workout logger dengan save set cepat
- copy previous set
- rest timer overlay
- workout summary
- progress overview + exercise detail chart
- body metrics manual
- settings inti
- export full backup JSON
- export CSV
- import full backup JSON
- restore workout aktif saat app reopen

## Out of Scope
- login/register
- sync cloud
- multi-device
- push notification kompleks
- social/community
- coach/client sharing
- wearable integration
- video exercise library
- custom theme builder
- advanced PR calculator screens
- AI recommendations

## MVP Success Criteria
- user bisa install dan langsung pakai tanpa akun
- routine pertama bisa dibuat < 2 menit
- save set terasa instan pada device mid-range
- seluruh data tetap ada setelah app restart
- backup JSON bisa di-export dan di-import kembali dengan utuh

## Release 1 Must-Have
- Home
- Routine List
- Create/Edit Routine
- Active Workout Logger
- Workout Summary
- Progress Overview
- Exercise Progress Detail
- Body Metrics
- Settings backup/import/export

## Release 1 Nice-to-Have

- mini chart pada body metrics

- seed routine examples
